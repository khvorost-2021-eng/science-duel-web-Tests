# SciDuel — Наука • Мотивация • Дуэли

SciDuel — это веб-приложение для проведения математических дуэлей в реальном времени с поддержкой игры с разных устройств. Проект объединяет стильный "космический" научный дизайн, систему аккаунтов и мультиплеер на базе WebSocket.

## 🚀 Особенности и реализованный функционал

1. **Дизайн и Интерфейс (Одиночная страница - SPA):**
   - Тёмная тема с эффектами "glassmorphism", градиентами и свечениями.
   - Анимированный фон с частицами (Canvas), создающий атмосферу науки.
   - Карусель с мотивационными цитатами великих учёных.

2. **Система пользователей (Локальная):**
   - Регистрация и авторизация (данные хранятся в `localStorage`).
   - Хеширование паролей (`btoa`).
   - Профиль пользователя: статистика побед, поражений, процента выигрышей.

3. **Мультиплеер в реальном времени (Node.js + Socket.io):**
   - **Серверная генерация примеров**: защита от читерства.
   - **Механика дуэли**: 3 уровня сложности (Лёгкий, Средний, Сложный).
   - **Варианты ответов**: 4 кнопки вместо ручного ввода.
   - **Настроечные комнаты (Rooms)**: Создание комнаты и подключение друга по 6-значному коду.
   - **Матчмейкинг (Поиск соперника)**: Автоматический поиск оппонента в очереди.
   - **Реванш**: Возможность сыграть с тем же игроком снова.

## 🛠 Технологический стек

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+), Canvas API.
- **Backend**: Node.js, Express, Socket.io.
- **Хранение данных**: `localStorage` (на клиенте), `Map/Array` в памяти (на сервере).

## 📥 Как запустить проект

Убедитесь, что у вас установлен **Node.js**.

1. Откройте терминал в папке проекта (`C:\Users\User\.gemini\antigravity\scratch\science-duel`).
2. Установите зависимости (если еще не установлены):
   ```bash
   npm install
   ```
3. Запустите сервер:
   ```bash
   npm start
   ```
4. В терминале появится сообщение с адресами:
   - `http://localhost:3000` (для открытия на текущем компьютере).
   - `http://192.168.x.x:3000` (IP-адрес для других устройств в **этой же Wi-Fi сети**).

## 🌐 Как играть с разных Wi-Fi (через Интернет)

По умолчанию сервер работает только в локальной сети. Чтобы играть с друзьями из других мест (с разным интернетом), вам нужно сделать сервер публичным.

**Самый простой способ (через Ngrok):**
1. Скачайте программу `ngrok` (https://ngrok.com/).
2. Пока запущен ваш сервер (`npm start`), откройте новую консоль и введите:
   ```bash
   ngrok http 3000
   ```
3. Ngrok выдаст публичную ссылку вида `https://xxxx-xxx.ngrok-free.app`.
4. Отправьте эту ссылку другу — и вы сможете играть вместе откуда угодно!

**Второй способ (Хостинг):**
Загрузить файлы проекта на бесплатный хостинг, поддерживающий Node.js (например, Render, Railway, Glitch).

## 📝 Структура проекта

- `server.js` — Серверная логика, WebSocket (Socket.io), комнаты, генерация примеров, таймер.
- `public/` — Папка со статическими файлами клиента:
  - `index.html` — Вёрстка всех экранов приложения.
  - `styles.css` — Стили (анимации, сетка арены, модальные окна, адаптивность).
  - `app.js` — Клиентская логика, взаимодействие с DOM, обработка событий Socket.io, система пользователей в localStorage.
- `package.json` — Зависимости проекта (express, socket.io).
